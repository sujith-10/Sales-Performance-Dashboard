# 📊 Sales Performance Dashboard — Superstore Dataset

![Dashboard Screenshot](Screenshot%202026-04-05%20184457.png)

## 📌 Project Overview

This project analyzes the **Superstore Sales Dataset** from Kaggle to uncover key business insights around sales performance, profitability, and customer trends. The goal was to build an end-to-end data analysis pipeline — from raw data cleaning to an interactive dashboard.

## 🔍 Key Analysis Performed

- 📦 **Sales by Category & Sub-Category** — Identified top-performing product lines
- 🌍 **Regional Sales Performance** — Compared revenue across regions
- 📈 **Monthly Sales Trend** — Tracked growth patterns over time
- 💰 **Profit Analysis** — Found most and least profitable products
- 👥 **Top Customers** — Identified high-value customers by revenue

---

## 💡 Key Insights

- **Technology** category generates the highest sales
- **West region** leads in overall revenue
- **Tables** sub-category shows consistent losses despite high sales
- Sales peak during **Q4** (October–December)
